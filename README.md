# Data-Science-portfolio
This is my data science portfolio showcasing my skills and ability to work with data to help make decisions and inferences.
This project was simply inspired by the sheer passion of wanting to learn and get better at this particular skill as I am working on transitioning to the world of data and the technologies that make it so easy to visualise data and lead decision making in industry. It's a continous project geared towards perfection.

The intial upload consists of the various  ways I have learnt to do data collection because my hope is that this  project follows an end to end data science life cycle. The first step of any data science life cycyle is data collection. 

I collected data using an Api from worldbank, used web scraping and collected it from the cloud.
